# Chunk 크기 조정

- 데이터를 일정 단위로 나눠 트랜잭션 범위 설정 / 청크 완료 시 커밋

크기가 작다 : 커밋 자주 발생 - 커밋 오버헤드로 인한 성능 저하, 메모리 사용 감소
크기가 크다 : 커밋 덜 발생 , 메모리 사용 증가
